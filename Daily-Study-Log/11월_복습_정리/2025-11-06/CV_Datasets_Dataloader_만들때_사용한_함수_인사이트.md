## 🧩 주요 함수 요약

### 🔹 문자열 및 경로 관련

| 함수           | 설명                     | 예시                                   |
| ------------ | ---------------------- | ------------------------------------ |
| `endswith()` | 문자열이 특정 문자로 끝나는지 확인    | `"data.csv".endswith(".csv") → True` |
| `exists()`   | 해당 경로(파일/폴더)가 존재하는지 확인 | `os.path.exists("/tmp") → True`      |

---

### 🔹 OpenCV 관련

| 함수               | 설명                      | 예시                                               |
| ---------------- | ----------------------- | ------------------------------------------------ |
| `cv2.imread()`   | 이미지를 **BGR 순서**로 읽음     | `image = cv2.imread(str(img_path))`              |
| `cv2.cvtColor()` | 색상 채널 순서 변경 (BGR → RGB) | `image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)` |

> 💡 OpenCV는 BGR로 이미지를 불러오므로,
> PyTorch·Matplotlib 같은 **RGB 기반 라이브러리와 맞추기 위해 변환** 필요.

### 🔹 Albumentations (데이터 증강)

| 규칙                                           | 이유                                     |
| -------------------------------------------- | -------------------------------------- |
| `if self.transform is not None:` 으로 증강 여부 확인 | transform이 정의된 경우에만 적용                 |
| `augmented = self.transform(image=image)`    | **딕셔너리 구조(`{"image": ...}`)** 로 전달해야 함 |
| `image = augmented['image']`                 | 반환 시에도 `'image'` 키로 접근해야 함             |

> 🧠 Albumentations의 `Compose()`는 입력/출력을 `dict` 형태로 다루기 때문.

---

### 🔹 정규표현식 활용

| 함수                                      | 설명             | 예시                            |
| --------------------------------------- | -------------- | ----------------------------- |
| `.str.contains('_aug\\d+', regex=True)` | 특정 패턴 포함 여부 확인 | `_aug1`, `_aug2` 같은 증강 데이터 탐지 |

> `regex=True` 덕분에 정규식 패턴(`\d+` = 숫자 1개 이상)을 인식함.
